## Development Setup

First, clone the repo and cd into the project:
```bash
git clone https://github.com/memogomez/api-rest.git
cd API
```

### Local setup (Linux & Windows)
Install dependencies:
```bash
npm install
```

### Run locally
```bash
npm start
```
